### WindSocket를 활용한 채팅 프로그램 구현
------------------------------------------------
작성자- 최윤수,정영진  

##### 서버측(Server.c)
1. 다수의 클라이언트 접속 관리
2. 각 클라이언트 메시지 수진 및 전파
3. 계정 등록 및 로그인 기능 구현

##### 클라이언트 측(client.c)
1. 서버에 접속하여 메시지 송수신
2. 계정 생성 및 로그인 기능 구현
3. 간단한 사용자 인터페이스 구현

### 느낀점  
-----------------------------------
소켓을 이용한 통신 구현 및 다중 클라이언트 관리에 대한 이해도가 높아짐을 느꼈습니다.
팀원간 협업이 원활하게 이루어짐에 따라 프로젝트 진행이 원활하게 되었고  
추후에도 C++언어를 활용한 다양한 시스템 프로그래밍을 수행해 보고싶습니다.
